laszlo weather application
demo for 176x208
ben shine 2.19.06

test.lzx shows the splash screen with a zip code entry field and an OK button.

results_test.lzx is a dummy application to show positioning, with hardcoded data. 

yahoo.lzx implements weather and forecast display using the yahoo xml api, hardcoded for
Providence, Rhode Island, zip code=02912. The yahoo weather api is documented at
http://developer.yahoo.net/weather/
