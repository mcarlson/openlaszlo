Animation

In Laszlo applications, anything can animate. Click the buttons to initiate simple animations. You can also drag the window to move it. All Laszlo components can be animated just like any view.

Animation can be used to orient the user when the application chages state. For a practical example of animating a window, check out the Dashboard demo, which uses animation in its window management. Dashboard windows have a UI control that allows a person to restore it to its original location, after it was moved or resized.
