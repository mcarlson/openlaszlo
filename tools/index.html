LZX Editing Tools

Editor support for LZX

Emacs Settings
Tells nXML and MMM Mode about LZX files. (Aquamacs includes both these packages by default.) Read the top of the file for installation instructions.
Vim Settings
Settings for the Vim text editor.
IDE for Laszlo
An Eclipse-based development environment. No longer an active Eclipse Foundation project.

Schemas to assist LZX source editing

lzx.dtd
A DTD for the LZX language.
lzx.rng
A RELAXNG schema for the LZX language.
lzx.rnc
A RELAXNG Compact schema for the LZX language.
lzx+libraries.rnc
A RELAXNG Compact schema for the LZX language plus the component libraries.
lzx+libraries-nons.rnc
A RELAXNG Compact schema for the LZX language plus the component libraries without a default namespace.
lzx.xsd
An XML Schema for the LZX language.
