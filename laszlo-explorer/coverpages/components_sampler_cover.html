Components

This sections demonstrates the Laszlo components. Frequently used components each have a page that shows multiple ways that the components can be parameterized to have a different appearance or behavior.

All Laszlo components can be specified declaratively, data driven and modified at run-time using JavaScript. The look of a component may be changes using style or by modifying component resources. For more about that see TODO: link in the designers guide.

All code for the components is available with the product in the /lps/components directory. There are 'lz' components which have the Laszlo look and feel and are showcases in this section. Every 'lz' component is a subclass of a corresponding 'base' component, which you can subclass to make custom components.
