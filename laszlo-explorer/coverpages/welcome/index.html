Welcome to OpenLaszlo 4.0.5

The open source platform for building rich internet applications

OpenLaszlo comprises a compiler for its LZX language and an optional server that provides proxying and transcoding. OpenLaszlo allows you to create web applications with a breakthrough user experience. It is offered for free under the business-friendly CPL license.

It's not just Flash anymore!

Before OpenLaszlo 4.0, OpenLaszlo applications were compiled in .swf format to run on Adobe's Flash Player. With OpenLaszlo 4.0 (OL4), we added the option of compilation to DHTML (browser-native JavaScript). OpenLaszlo 4 handles browser idiosyncrasies so you don't have to.

OpenLaszlo 4.0.5 is the recommended platform for .swf development

With OpenLaszlo 4.0.5, there is no longer any reason to continue application development using OpenLaszlo 3.4. Support for DHTML applications remains at "beta" level; it will be fully supported with OL 4.1. This release has been tested by users world wide and vetted by the Laszlo Systems Quality Assurance team. Hundreds of improvements have been made since OpenLaszlo 4.0. Please see the release notes for a detailed description of new features in this release, changes since the last major release, and known problems.

OpenLaszlo Depends on You

As an open source project, we rely on the goodwill of our users. You can support OpenLaszlo by filing bug reports, sending us feedback, contributing bug fixes and enhancements, and helping us to spread the word.
