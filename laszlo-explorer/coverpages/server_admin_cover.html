Server Admin

The "Admin Console" offers a graphical user interface for accessing server commands. This is written entirely in LZX using standard OpenLaszlo components.

Server commands can be executed directly using adminstrative 'lzt' request types. Some common commands are listed below. You can click the links to execute the commands.

  • Clear cache
  • Server statistics
  • Cache statistics
  • Cache details
  • OpenLaszlo ServerError count
  • Clear OpenLaszlo Server error count
  • OpenLaszlo Server server info
  • Current OpenLaszlo Server log file (can be long)

This distribution is configured for development, for deployment details see Deploying OpenLaszlo Applications in the System Administrator's Guide.
