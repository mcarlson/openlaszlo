Overview of Laszlo Explorer

The Laszlo Explorer provides the following resources to help you learn how to design, develop and deploy Laszlo applications:

Demos: rich internet applications that show unity of design

Laszlo in Ten Minutes: an interactive guided tour to get you started

Small Applications: shorter programs that illustrate specific aspects of Laszlo development

Components: an illustrated compendium of Laszlo's prebuilt, user-extendable objects

Documentation: tutorials, expository, and reference—including scores of live, editable examples

The Laszlo Explorer also provides convenient access to tools for administering the Laszlo Presentation Server. These tools are intended for use by advanced developers.
