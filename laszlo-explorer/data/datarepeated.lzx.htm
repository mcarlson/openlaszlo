Notes                               for more details  datapath reference
If the datapath of a component points to a node with multiple instances, the component is automatically replicated to display the value of each node. This is an easy way of displaying lists. You can use XPath syntax to limit the nodes you want to see. For example, to display the second node only, change the datapath attribute of the view from dset:/phonebook/employee to dset:/phonebook/employee[2], then press the Update button. The application is recompiled and you can see the results of your changes in the running application area.
