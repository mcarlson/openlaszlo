Original version
----------------
This translation is based on the English documentation from the 3.2 release. 


Installation
------------
To install the French translation of the guide and overwrite the English version : 
- Replace HTML files from the folder 'OpenLaszlo Server 3.x/Server/lps-3.x/docs/guide' with the attached HTML files. 
- Open the translated guide using laszlo-explorer or using the following URL : http://127.0.0.1:8080/lps-3.2/docs/guide/'.

To install the french translation of the guide without overwriting the english version : 
- Copy the folder 'OpenLaszlo Server 3.x/Server/lps-3.x/docs' and name it 'OpenLaszlo Server 3.x/Server/lps-3.x/docs_fr',
- Replace HTML files from the folder 'OpenLaszlo Server 3.x /Server/lps-3.x/docs_fr/guide' with the attached HTML files. 
- Open the translated Guide using the following URL : 'http://127.0.0.1:8080/lps-3.2/docs_fr/guide/'
