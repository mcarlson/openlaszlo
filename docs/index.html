OpenLaszlo Documentation

OpenLaszlo's developer documentation includes overviews, tutorials, and details on application development with Laszlo's technology. Follow the links below to find specific documentation.

  • User's Guides
    Materials for adopters of the OpenLaszlo platform.
    • Release Notes
      What's new in the release; changes to LZX syntax.
    • Installation Instructions
      How to install and upgrade the OpenLaszlo Server.
    • OpenLaszlo Application Developer's Guide
      In depth explanations of OpenLaszlo concepts.
    • OpenLaszlo System Administrator's Guide
      How to make OpenLaszlo applications available on the web.
    • OpenLaszlo Reference
      A complete listing of every public LZX tag and API -- including tag attributes and API methods and parameters. (The link above takes you to the OpenLaszlo 3.4 reference on www.openlaszlo.org. OpenLaszlo 4 Reference is also available locally, but it is not as polished as the 3.4 reference. We recommend you use the OpenLaszlo 3.4 Reference.)
    • OpenLaszlo Components Guide
      An Overview of the OpenLaszlo Component Set for User Interface Designers.
    • Master Index
      A complete index covering the Developer and Adminstrator's Guides as well as the OpenLaszlo Reference.
  • Contributor's Guide
    Materials for contributors to the platform.
    • Comprehensive OpenLaszlo Reference
      A comprehensive reference including public and private APIs, including kernel interfaces.

If you cannot find answers here, please ask your question on the forums. Comments on the documents themselves can be posted to the forums as well, or filed in the OpenLaszlo bug database.
