Source Fingerprints

The goal of source fingerprints is to increase the chance that we can prove that a copy of our source code was copied from our source code. This is done by adding nonfunctional code to the sources: methods to the LPS, and comments to the LFC. (The LFC changes are susceptible to comment stripping, but are constrained by the fact that we don't want to increase the LFC size.)

LPS Methods

These are non-functional, but don't remove them:

  • Canvas.getISBN
  • FontFamily.getBookRecommendations
  • SplashCompiler.getBlogList

LFC Comments

These look pointless, but don't remove them:
