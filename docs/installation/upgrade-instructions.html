Upgrading the OpenLaszlo Server (formerly called Laszlo Presentation Server)

To upgrade from any previous release to @VERSIONID@:

  1. Read the release notes.
  2. Stop the LPS if it is already running.
  3. Run the @VERSIONID@ installer.
  4. Copy any LZX files from your my-apps directory in the previous release the new my-apps directory in @VERSIONID@.
  5. You can safely delete the entire lps-2.0p, lps-v1 or lps-1.0.x directory.
