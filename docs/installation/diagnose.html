Diagnosing Laszlo Presentation Server Developer Edition Installation

This tutorial provides some suggested steps to try in case you are having problems running the Laszlo applications bundled with the installation.

Symptom

Try
Page not found error.

Tomcat is not running. Start Tomcat
   

If a service other than Tomcat or another version of Tomcat is running on port 8080, it would prevent Tomcat from running your Laszlo applications. Stop the service from running and then try to run your Laszlo application again.
