Colors and .SWF files

Chris from Semaphore is porting some old code that used to colorize a SWF. He asserts that it's not just tinting he's looking for: ================================================= <view name="some_color" resource="colorbox.swf"> <method name="applyColor" args="someRGB"> var thisClass = this.parent; this.colorControl=new Color(this.movieClipRef); this.colorControl.setRGB(someRGB); </method> </view> ================================================= a) Does that code really affect the SWF or is it the same as setcolortransform? LzView.setColor is probably what he wants. It does the same thing as the example code you have (takes an RGB and applies it as an offset.) LzView.setColorTransform allows you to set both the RGB offsets and the RGB alphas. b) What's the movieClipRef for? That's the way Flash does it. To control the color of a clip you make a Color object that references the clip. It's like the Color object exposes some otherwise inaccessible API. Here's the post: http://www.laszlosystems.com/developers/community/forums/showthread.php postid=3315#post3315

<canvas> <view width="60" height="60" bgcolor="red" onclick="changecolor()"> <method name="changecolor"> this.setattribute(bgcolor, "red") </method> </view> </canvas> <canvas debug="true"> <simplelayout axis="y" spacing="10" /> <view id="foo" width="400" height="200"> <simplelayout axis="y" spacing="10" /> <text> <font color="#0000FF"> Hello world </font> </text> </view> <button>Apply tint now <handler name="onclick"> var transColor = {rb:255}; foo.setColorTransform( transColor ); </handler> </button> </canvas> SetColorTransform applies a tint :-) SetColor sets an rgb color, but setColorTransform takes an argument with 8 components that lets you adjust the percentage and offset of each of the red/green/blue channels or an overall percent and offset. It is applied as a transform to the view and all subviews. I'm no color expert, so I can't tell you how to compute the right value for a particular tint, but I'm guessing that if you wanted to tint something red, you could just apply a transform with a positive red offset.
