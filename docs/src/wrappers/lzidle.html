Service Event Idle service. api-lzidle.xml

The LFC will generate idle events at some fixed, unspecified frequency. Here is a simple example that shows how to receive the onidle event:

<canvas height="140" debug="true"> <attribute name="counter" value="0"/> <method name="onidle" event="onidle" reference="LzIdle"> while (counter &lt; 4) { counter++; Debug.write("idling " + counter); } </method> </canvas>
