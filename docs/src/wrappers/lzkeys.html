Service Event Keyboard input service. api-lzkeys.xml

This service handles keyboard input events.

Here is a simple example:

<canvas height="140" debug="true"> <method event="onkeydown" reference="LzKeys" args="k"> Debug.write("key " + k + " down"); </method> <method event="onkeyup" reference="LzKeys" args="k"> Debug.write("key " + k + " up"); </method> <method name="pressA"> Debug.write("A pressed"); </method> <method event="oninit"> del = new LzDelegate(this, "pressA"); LzKeys.callOnKeyCombo(del, ["A"]); </method> </canvas>
