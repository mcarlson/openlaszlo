The debug tag controls the initial appearance and options of the debugger when debugging is enabled. Debugging is enabled either by including the debug="true" option on the canvas tag, or adding the ?debug=true query argument to the application's URL.

<canvas debug="true" height="120"> <debug width="100" height="50" x="5" y="5" fontsize="8"/> <script> Debug.write("Hello world!"); </script> </canvas>

The debug tag controls the appearance of the debugger when debugging is on. It does not cause the application to be compiled with debugging mode enabled. Only the debug option on the canvas and the debug query parameter of the URL do this.

For more information, see the Debugging chapter of the Guide.
