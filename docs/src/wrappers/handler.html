Language Feature Scripting Attaches an event handler to an object or class. tag-handler.xml

Attaches a handler method for the named event to the object that contains this element. The method must have a name

The name attribute names the event for which this handler should be invoked.

Note: events can be declared via the event tag.

For example, if a handler is defined via:

<view id="obj"> <handler name="onmyevent" args="a"> this.setBGColor(0xff0000); Debug.write(this,'got onblah event, arg is',a); </handler> </view>

then script code obj.onmyevent.sendEvent(259) will invoke this handler method.
