Language Feature Scripting Attaches a function or event handler to an object or class. tag-method.xml

Attaches a method to the object that contains this element. The method must have either a name or event attribute.

If the method has a name attribute, it defines a method that can be invoked from JavaScript with this name. For example, if a method is defined via:

<view id="obj"> <method name="f" args="a, b"> return a+b; </method> </view>

then script code can invoke obj.f(1, 2) to add two numbers.

If the method has an event attribute, the script will be executed when the referenced view receives an event with this name. For example,

<view> <method event="onclick"> f(this); </method> </view>

defines code that applies the function f to the view object, when the view is clicked on.
