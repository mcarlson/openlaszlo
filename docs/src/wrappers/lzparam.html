LFC Data A utility class for manipulating name-value pairs api-lzparam.xml

LzParam provide utilities for manipulating name-value pairs. For example, LzParam objects can be used to manage a name/value pairs for an HTTP request.

<method name="sendData" args="action"> var d=canvas.datasets.dsSendData; var p=new LzParam(); p.addValue( "action", action, true); p.addValue( "firstName", "Bob", true); p.addValue( "lastName", "Jones", true); p.addValue( "phone", "415-255-2392", true); p.addValue( "email", "bob@jones.com", true); d.setQueryString( p ); d.doRequest(); </method>
