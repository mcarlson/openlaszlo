LFC Data A utility class for manipulating URLs api-lzurl.xml

LzURL is used to break URLs into component parts: protocol, host, port, path, file and query. Each part can be changed independently, and a new URL created. For example:

<canvas height="220" debug="true"> <debug height="200" width="450"/> <script> var urlstring = 'http://localhost:8080/lps/lzurl_example.lzx?lzt=swf'; var url = new LzURL(urlstring); // list each part of this url Debug.write("protocol: " + url.protocol); Debug.write("host: " + url.host); Debug.write("port: " + url.port); Debug.write("path: " + url.path); Debug.write("file: " + url.file); Debug.write("query: " + url.query); // list the url Debug.write("Complete URL: " + url); // Make it secure url.protocol = 'https'; url.port = 8443; Debug.write("Secure URL: " + url); // Change the query string url.query = 'lzt=source'; Debug.write("New path: " + url); </script> </canvas>
