LFC View The most basic viewable element. tag-view.xml api-lzview.xml

The view is the most basic viewable element in a Laszlo application. Anything that is displayed on the canvas is a view or extends the view class. A view is a rectangle, which can be visible or invisible, and which can contain other views or display "resources" such as images, .swf files, or other supported media. One view controls one displayable resource. The view system is strictly hierarchical; that is, each view has a single parent but can have multiple children.

The following code shows one view nested inside another:

<canvas height="50" width="500"> <view width="50" height="50" bgcolor="red"> <view width="30" height="30" bgcolor="blue"/> </view> </canvas>

LzView extends LzNode, which is the fundamental abstract class of LZX.

For more a gentle introduction to views please see the Laszlo tutorials, particularly Laszlo Basics and Understanding Views. For a more rigorous explanation of views see theDeveloper's Guide, particularly the views chapter.
