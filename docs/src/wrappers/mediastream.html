Audio Video audio video A connection to an RTMP server. extensions/av/mediastream.xml

A mediastream opens a connection for recieving streaming media.

Note: Total time is only defined for FLV's delivered over http if they have "metaData" defined, and we don't know the stream length until it starts playing.

<canvas> <simplelayout axis="y" inset="10" spacing="5"/> <constantlayout axis="x" value="10"/> <videoview> <mediastream id="s" url="http://localhost:8080/video/test/video/videos/good-dog.flv" debug="true" /> </videoview> <button text="play" onclick="s.play()" /> <button text="pause" onclick="s.pause()" /> <text text="${'playtime: ' + s.playtime}" /> <text text="${'totaltime: ' + s.totaltime}" /> <text width="300" multiline="true" > </text> </canvas>
