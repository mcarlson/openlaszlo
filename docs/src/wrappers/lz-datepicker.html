LZ Components view A simple date picker lz/datepicker.xml

Datepicker is a simple component that makes it easy for users to select dates.

<canvas height="200"> <datepicker showingdate="new Date(2004,3,1)" earliestdate="new Date( 2003, 9, 5)" latestdate="new Date( 2005, 1, 5)" selecteddate="new Date( 2004, 3, 20)"> <method event="onselecteddate"> if( this.selecteddate != null ) { display.year.setText( this.selecteddate.getFullYear() ); display.month.datapath.setXPath( "datepicker_strings_en:/months/month[@index='" + this.selecteddate.getMonth() + "']/@full" ); display.date.setText( this.selecteddate.getDate() ); } </method> </datepicker> <view id="display"> <text name="month" resize="true" datapath="."/> <text name="date" resize="true"/> <text name="year" resize="true"/> <simplelayout axis="x" spacing="2"/> </view> <simplelayout axis="y" spacing="10"/> </canvas>
