LZ Components View A draggable panel that provides common behavior for various window classes. lz/windowpanel.xml

The windowpanel tag creates a floating view that has a title area and can be moved by the user. It provides common behavior for window, modaldialog, and alert.

<canvas width="300" height="150"> <windowpanel x="10" y="10" width="200" height="100" title="windowpanel" closeable="true"> <text>Hello</text> </windowpanel> </canvas> window modaldialog alert
