Debugger

Core and UI Debugging

The core debugger is defined in LzDebug.as. This file is compiled into the debug versions of the LFC, and defines the Debug class.

LZX applications that are compiled with debugging (via the &debug query parameter, or <canvas debug="true">) link against a debug version of the LFC, and also include the debugger LZX component. This file replaces many of the methods on Debug with functions that use provide more of a user interface. Many of these functions call the core debug functions, whose names are also preserved as global variables.
